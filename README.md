# Image Slider Project

## Overview
A simple interactive image slider using HTML, CSS, and JavaScript.
The slider uses background images for the container and allows navigation using left/right buttons. The slider loops infinitely.

## Features
- Left/Right navigation buttons
- Infinite loop slider
- Background image updates dynamically
- Responsive and styled container
- Easy to extend with more images

## How to Use
1. Clone the repository
2. Place your images (`img1.jpg` to `img5.jpg`) in the same folder
3. Open `index.html` in the browser
4. Click the buttons to navigate the images
